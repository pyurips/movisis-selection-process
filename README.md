# Readify (e-commerce de e-books)

## Requisitos

- Node.js versão 18+¹
- NPM, Yarn, PNPM ou Bun como gerenciador de pacotes

## Comandos

Para instalar as dependências do projeto, execute:

```bash
npm install
# ou
yarn
# ou
pnpm install
# ou
bun install
```

Para iniciar o servidor de desenvolvimento, execute:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm run dev
# ou
bun run dev
```
